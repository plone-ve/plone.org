# ploneorg

The main destination for Plone

## Features

### Content Types

- A Foundation sponsor.

- Plone Foundation Member.

- A Plone release.


### Initial content

This package contains a simple volto configuration.

Installation
------------

Install ploneorg with `pip`:

```shell
pip install ploneorg
```
And to create the Plone site:

```shell
make create_site
```
